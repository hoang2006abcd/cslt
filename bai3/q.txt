Cho một số nguyên d biểu diễn tổng số giây.

Hãy chuyển d thành định dạng gồm giờ (h), phút (m), giây (s).

     Input
Một dòng chứa một số nguyên duy nhất d — tổng số giây.

Giới hạn: 1 ≤ d ≤ 10^18

     Output
In ra 3 số nguyên không âm:
h m s
— lần lượt là số giờ, số phút và số giây tương ứng.


test 1:
    input : 3929
    output : 1 5 29

test 2:
    input : 3600
    output : 1 0 0 

test 3 :
    input : 60
    output : 0 1 0
